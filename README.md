# Double Pendulum Playground

Interactive Python experiments for a double pendulum.

## Files

- `double_pendulum.py`
  - 2D double pendulum animation with trace rendering.
- `double_pendulum_3d.py`
  - 3D double pendulum viewer with mouse rotation and zoom.
- `double_pendulum_poincare.py`
  - Poincare section viewer for comparing regular and chaotic motion.
- `double_pendulum_chaos_map.py`
  - FTLE-based chaos heatmap over the `theta1-theta2` plane.

## Requirements

- Python 3.11 or later
- `numpy`
- `tkinter`

## Run

```powershell
python double_pendulum.py
python double_pendulum_3d.py
python double_pendulum_poincare.py
python double_pendulum_chaos_map.py
```
